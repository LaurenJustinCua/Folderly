# Folderly – File Organizer (Proof of Concept)

Folderly is a web-based file organizer application developed as an academic proof of concept. 
The project focuses on allowing users to upload, organize, and manage files through a simple web interface.

⚠️ Note: This project is not fully complete. During the final stages of development, multiple technical issues and runtime errors occurred close to the submission deadline, which prevented full feature completion and refinement.

---

## Project Purpose
The purpose of this project is to demonstrate backend development concepts including:
- File uploads and storage
- User authentication concepts
- Database interaction
- Server-side routing

This project was created for educational purposes and is not intended for production use.

---

## Technologies Used
- Node.js
- Express.js
- SQLite
- Multer (file uploads)
- JSON Web Tokens (JWT)
- HTML, CSS, JavaScript

---

## Current Features
- Basic backend server setup
- File upload handling
- Directory-based file storage
- SQLite database integration
- REST API structure

---

## Known Limitations
- Incomplete authentication flow
- File upload inconsistencies
- Frontend integration issues
- Error handling not fully implemented
- Not all planned features were finalized
